# Human Safe List

## Project Description
Human Safe List is a project designed to maintain a curated list of safe entities, individuals, and resources to protect and assist communities in need. The goal is to create a tool that can be utilized by various organizations to promote safety and awareness.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/kasipora/human-safe-list.git
   ```
2. Navigate into the project directory:
   ```bash
   cd human-safe-list
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Features
- Curated lists of safe entities
- Search and filter functionality
- User submission for new entries
- Community feedback on listings
- Export lists in various formats (CSV, JSON)
